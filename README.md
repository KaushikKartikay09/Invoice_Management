# Invoice_Management
This is a simple Invoice Management System built using core Python and executed on Python IDLE.
The project is designed for beginners to practice Python fundamentals by implementing a basic billing system using a menu-driven program.
It allows users to create, store, and view invoices using basic concepts like functions, loops, and file handling.

**Features**

Create new invoices
Store customer and item details
Calculate total invoice amount
View saved invoices
Simple menu-driven interface
File handling for data storage

**Technologies Used**

Programming Language: Python
IDE: Python IDLE

**How to Run the Project**

Install Python (version 3.x recommended)
Open Python IDLE
Open the invoice.py file
Run the program using F5
Follow the on-screen menu options

**Concepts Used**

Variables and Data Types
Conditional Statements
Loops
Functions
File Handling

**Purpose of the Project**

This project is created to:

Strengthen understanding of core Python concepts
Learn how real-world problems like invoice management can be solved using basic programming
Serve as a beginner-level academic or practice project
